


Checklist para preparação do Ambiente de Desenvolvimento:
link para o checklist no Github - https://github.com/luizsodrerj/treinamento/tree/master/spring-backend-exemplo
clique no arquivo notas.txt para obter todos os passos do checklist 

Passo 1: 
Instalar o JDK 17 ou superior. 
- link: https://www.oracle.com/java/technologies/downloads/#java17
- crie a variavel de ambiente JAVA_HOME no Sistema Operacional apontando para o home da instalação do JDK

Passo 2:
- Criar o Banco de Dados de teste. 
- Baixe a biblioteca do HsqlDB (link - https://sourceforge.net/projects/hsqldb/files/hsqldb/hsqldb_2_2/)
  e inicie-o em modo Server: 
- Iniciar HsqlDB em modo server:
- Cria uma pasta e copie para ela o jar do Banco.
- Dentro da pasta, abra um prompt de comando e digite os seguintes comandos: 
	set CLASSPATH=%CLASSPATH%;hsqldb-2.2.9.jar
	java org.hsqldb.Server -database cliente
- Use a ferramenta DBeaver para criar a tabela de teste (link - https://dbeaver.io/download/)  
- Script DDL de criação da tabela de clientes:
CREATE TABLE CLIENTES (
	ID_CLIENTE INTEGER NOT NULL IDENTITY,
	NOME_RAZAO VARCHAR(300),
	DT_CADASTRO DATE,
	CONSTRAINT SYS_PK_ID_CLIENTE PRIMARY KEY(ID_CLIENTE)
);

Passo 3:
- Instale a ferramenta Maven (link - https://maven.apache.org/)
- configure a pasta bin no path do Sistema

Passo 4:
- Instale a IDE IntelliJ IDEA Community Edition (link - www.jetbrains.com/idea/download/)

Passo 5:
- Use a ferramenta Spring initializer para criar o primeiro projeto Backend de exemplo
(link: https://start.spring.io/)
